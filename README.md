# App-music
Api App music en Node js

- API REST DE PLAYLIST Y CANCIONES 
- EN NODE EXPRESS-TYPESCRIPT

Primero debemos crear el archivo .env
- DATABASE_URL="file:./dev.db"
- PORT=5000


Como Usar:

- npm install

- npx prisma migrate dev --name init

- npm run dev
